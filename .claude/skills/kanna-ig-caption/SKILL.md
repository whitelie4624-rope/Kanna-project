# KANNA Instagram キャプションSkill

## 概要
KANNAのInstagram投稿に最適なキャプションとハッシュタグセットを生成するSkillです。女風らしい上品な言葉遣いで、癒し・非日常・信頼感を訴求する200〜400字のキャプションと20〜30個のハッシュタグを生成します。

## KANNAについて
- **サービス種別**：女性用風俗（女風）・リラクゼーション
- **ターゲット**：20〜40代女性
- **場所**：東京
- **SNS**：Instagram・X（旧Twitter）・note
- **トーン**：上品・温かみ・信頼感・非日常感
- **セラピスト**：男性セラピスト（オーナー兼任）

## 手順

### Step 1：投稿内容の確認
以下の情報を確認する：
- 投稿する写真・動画の内容（空間 / アロマ / 施術アイテム / 日常風景など）
- 投稿のテーマ・目的（認知拡大 / 雰囲気訴求 / キャンペーン告知 / 日記）
- 伝えたいメインメッセージ（1つに絞る）
- 投稿のタイミング（平日 / 週末 / イベント前後）

### Step 2：キャプション構成の設計
以下の構成でキャプションを生成する：
1. **書き出し（フック）**：最初の1〜2文で読者を引き込む（50〜80字）
2. **本文**：テーマに沿った世界観・情景描写（100〜200字）
3. **締め**：読者への語りかけ・行動喚起（50〜80字）

### Step 3：KANNAのトーンで執筆
- 直接的な性的表現は一切使わない
- 「癒し」「安らぎ」「ひととき」「寄り添う」などのキーワードを自然に使う
- 顧客（読者）に語りかける親しみやすい文体
- Instagramらしい改行・空白行を適切に使う

### Step 4：ハッシュタグの設計
以下の3カテゴリからバランスよく20〜30個選ぶ：
- **大カテゴリ（5〜8個）**：広いリーチを狙う一般的なタグ
  例：#女性用風俗 #リラクゼーション #東京 #癒し #ご褒美
- **中カテゴリ（8〜12個）**：ターゲットに刺さる中程度のタグ
  例：#女風 #東京リラクゼーション #30代女性 #自分へのご褒美
- **ブランドタグ（2〜5個）**：KANNA独自のタグ
  例：#KANNA #kanna女風 #KANNArelax

## 出力フォーマット

```
## Instagram投稿キャプション

### キャプション本文
[200〜400字のキャプション]

### ハッシュタグ（[個数]個）
#タグ1 #タグ2 #タグ3 ...

---
推奨投稿時間：
投稿タイプ：[日記 / 施術紹介 / 空間 / キャンペーンなど]
補足メモ：
```

## 注意事項
- 「女風」「風俗」の文字は使い方に注意（InstagramのアルゴリズムがNGとする場合がある）
- 性的な内容・過激な表現は絶対に使わない
- 具体的な料金・プラン名はキャプションに含めない（プロフィールリンクに誘導）
- 他の顧客・人物を特定できる情報は含めない
- ハッシュタグは英語・日本語を混在させると効果的
