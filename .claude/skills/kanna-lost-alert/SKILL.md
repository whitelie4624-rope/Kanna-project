# KANNA 失客フォローアップSkill

## 概要
一定期間来店のない顧客への再来店促進メッセージを生成するSkillです。自然な気遣い→近況確認→予約誘導の構成で150字以内のメッセージを生成します。来店間隔（1ヶ月・3ヶ月・半年）別に文体を変え、義務感を感じさせない文体にします。

## KANNAについて
- **サービス種別**：女性用風俗（女風）・リラクゼーション
- **ターゲット**：20〜40代女性
- **場所**：東京
- **トーン**：上品・温かみ・信頼感・非日常感
- **重視すること**：リピート率・顧客との信頼関係・ブランド一貫性

## 手順

### Step 1：状況の確認
以下を確認する：
- 最終来店からの経過期間（1ヶ月 / 3ヶ月 / 半年 / それ以上）
- 最後に来店した際の様子（カルテがあれば参照）
- 特別なきっかけがある場合（季節の変わり目・キャンペーン・新コース開始など）

### Step 2：来店間隔別のアプローチ設計
**1ヶ月経過：**
- まだ心理的距離は近いタイミング
- 「お体の調子」への自然な気遣いベース
- 積極的な予約誘導は控えめに

**3ヶ月経過：**
- 少し間が空いてしまったことを認識している
- 「最近どうかな」という自然な近況確認
- 新しいコースや季節の話題を絡めると自然

**半年以上経過：**
- 相手もこちらのことを忘れかけている可能性
- 無理に「来てください」とせず、「思い出してもらえたら」ニュアンス
- 再来店へのハードルを極力下げる

### Step 3：メッセージ生成
以下の構成で150字以内のメッセージを生成する：
1. **自然な気遣い**（30〜50字）：体調・近況への気遣い
2. **近況確認 or 近況報告**（50〜70字）：KANNA側の近況や季節ネタを絡める
3. **予約誘導**（20〜40字）：「いつでも」スタンスで自然に

### Step 4：バリエーション提案
3パターン提案する：
- メッセージA：気遣いベース（ソフト）
- メッセージB：新情報・季節ネタを絡めたもの（フック付き）
- メッセージC：非常に簡潔なもの（最もプレッシャーが少ない）

## 出力フォーマット

```
## 失客フォローアップDM（最終来店から[期間]）

### メッセージA（気遣いベース）
[150字以内のメッセージ]
字数：○○字

### メッセージB（季節・新情報フック）
[150字以内のメッセージ]
字数：○○字

### メッセージC（簡潔版）
[150字以内のメッセージ]
字数：○○字

---
推奨送信タイミング：
文体のポイント：
注意点：
```

## 注意事項
- 「なぜ来ないのですか」「ずっと待っています」などの責め感・プレッシャーは絶対NG
- 返信しないことへの言及もしない（「返信なかったけど」など）
- 施術の詳細な内容には触れない
- 顧客の個人的な事情・悩みへの言及は最小限にする
- 来店を「お願い」ではなく「いつでもウェルカム」のスタンスで表現する

## 出力例

```
## 失客フォローアップDM（最終来店から3ヶ月）

### メッセージA（気遣いベース）
こんにちは。お変わりありませんか？
少し肌寒い季節になりましたね。
お体、ちゃんと休めていますか？
またいつかご都合のよいときに、ゆっくりしに来てください。
お待ちしています。
字数：80字

### メッセージB（季節フック）
こんにちは。秋も深まってきましたね。
疲れが出やすい季節でもあるので、お体のことが少し気になっていました。
新しいアロマも取り入れたので、気が向いたらぜひ。
お待ちしています。
字数：87字

### メッセージC（簡潔版）
お久しぶりです。お元気でしょうか。
また会える日を楽しみにしています。
字数：36字

---
推奨送信タイミング：季節の変わり目・連休前
注意点：同じ方への再送は最低2ヶ月以上空けること
```
