# KANNA 顧客カルテSkill

## 概要
施術後の顧客情報をカルテ形式で自動整理するSkillです。基本情報・施術内容・顧客の反応・次回提案・備考を含む顧客カルテを生成し、次回施術の提案まで含めます。

## KANNAについて
- **サービス種別**：女性用風俗（女風）・リラクゼーション
- **ターゲット**：20〜40代女性
- **場所**：東京
- **重視すること**：リピート率・顧客との信頼関係・ブランド一貫性
- **セラピスト**：男性セラピスト（オーナー兼任）

## 手順

### Step 1：施術後データの入力受付
以下の情報をユーザーから入力してもらう（任意入力）：
- 来店日・来店時間
- 顧客識別子（名前は不要。「〇〇さん」「Aさん」など任意の呼称でOK）
- 来店回数（初回 / 2回目 / ○回目）
- 選択したコース・時間
- 顧客の状態（体の疲れ具合・気分・特に気になっている部位など）
- 施術内容・アプローチのメモ
- 顧客の反応・様子（リラックスした / よく話してくれた / 眠っていたなど）
- 特記事項・気になったこと
- 次回のご希望（言っていた場合）

### Step 2：カルテの構造化
入力された情報を以下のカルテ形式に整理する：
- 読み返したときに施術の流れが思い出せる構成
- 次回来店時に活かせる情報を前面に出す

### Step 3：次回施術の提案
今回の施術内容と顧客の反応をもとに、次回施術への提案を生成する：
- おすすめのコース・時間
- 重点的にアプローチすべき部位・テーマ
- 次回来店の理想的なタイミング（「2〜3週間後がおすすめ」など）

### Step 4：フォローアップDMのメモ
次回来店促進のためのフォローアップタイミングと一言メモを生成する

## 出力フォーマット

```
# 顧客カルテ

## 基本情報
- 来店日：
- 顧客識別子：
- 来店回数：
- コース：

## 施術記録
### 来店時の状態
- 体の状態：
- 気になる部位：
- 気分・テンション：

### 施術内容
- 主なアプローチ：
- 特に重点を置いた箇所：
- 時間配分のメモ：

### 顧客の反応・様子
-

## 特記事項・覚えておきたいこと
-

## 次回への提案

### 次回おすすめコース・アプローチ
-

### 次回来店の理想タイミング
-

### 次回話せると良いこと・聞けること
-

## フォローアップメモ
- DM送信タイミング：（例：来店から10日後）
- DM一言メモ：（次回DMで触れると喜ばれそうなこと）
```

## 注意事項
- 顧客の個人情報（フルネーム・連絡先・住所）は入力しない・記録しない
- 施術の詳細な性的内容は記録しない（「心地よい施術を提供した」などの
  抽象的な表現にとどめる）
- カルテデータは適切に管理し、外部に漏洩させない
- 顧客が話してくれたプライベートな話は「慎重に扱うべき情報」として
  最小限の記録にとどめる

## 出力例

```
# 顧客カルテ

## 基本情報
- 来店日：2024年4月15日（月）19:00
- 顧客識別子：Sさん
- 来店回数：3回目
- コース：90分スタンダード

## 施術記録
### 来店時の状態
- 体の状態：肩・首まわりの張りが強め
- 気になる部位：肩甲骨まわり
- 気分：「仕事疲れ」とおっしゃっていた。来店時は少し緊張気味

### 顧客の反応・様子
- 施術後半はほぼ眠っていた
- 終わり際に「すごく楽になった」と笑顔
- 「また来月来たい」とおっしゃっていた

## 次回への提案
- おすすめコース：90分（今回と同様）
- 重点アプローチ：肩甲骨まわり継続 + 腰まわりも確認する
- 理想タイミング：3〜4週間後（5月上〜中旬）

## フォローアップメモ
- DM送信：来店から10日後（4月25日頃）
- DM一言：肩の調子が気になるため、体の状態を気遣う文面にする
```
