# KANNA 日記・ブログ記事Skill

## 概要
KANNAの世界観に合った日記・ブログ記事を生成するSkillです。セラピスト目線で、上品で温かみのあるトーンの800〜1200字のテキストを生成します。施術の詳細な性的表現はNGで、非日常感・癒しを中心に表現します。

## KANNAについて
- **サービス種別**：女性用風俗（女風）・リラクゼーション
- **ターゲット**：20〜40代女性
- **場所**：東京
- **SNS**：Instagram・X（旧Twitter）・note
- **トーン**：上品・温かみ・信頼感・非日常感
- **セラピスト**：男性セラピスト（オーナー兼任）

## 手順

### Step 1：記事テーマの確認
以下の情報を確認する：
- テーマ（今日の出来事 / 施術の雰囲気 / 季節ネタ / セラピスト自身の気づき）
- 投稿先（Instagram / note / ブログ）
- 特に伝えたいこと・エピソードの概要（箇条書きでもOK）
- 主人公の視点（セラピスト自身 / KANNAの空間そのもの）

### Step 2：記事の構成設計
800〜1200字の記事を以下の構成で書く：
1. **書き出し（導入）**：読者の心を掴む一文・情景描写（150〜200字）
2. **本文①：エピソード**：今日の出来事・気づきを具体的に（300〜400字）
3. **本文②：想い・気づき**：セラピストとしての想いや信念（200〜300字）
4. **締め**：読者への語りかけ・次回予告（100〜150字）

### Step 3：KANNAのトーンで執筆
以下のトーンを徹底する：
- **上品さ**：丁寧語・品のある言葉選び
- **温かみ**：読者に寄り添う、語りかけるような文体
- **非日常感**：日常を離れた特別な時間・空間の演出
- **信頼感**：セラピストとしての誠実さ・専門性が伝わる表現

### Step 4：表現の確認（NG事項の排除）
以下を含まないか確認する：
- 施術の詳細な性的表現
- 顧客の個人情報・来店エピソードの特定
- 誇大な効果の訴求（医療的効果など）

## 出力フォーマット

```
【タイトル】
[記事タイトル（20〜30字）]

【本文】
[800〜1200字の記事本文]

【投稿メモ】
- 最適な投稿タイミング：
- おすすめハッシュタグ（note・Instagramの場合）：
```

## 注意事項
- 施術の詳細な性的内容・行為の具体的描写は絶対にしない
- 顧客の個人情報・来店日・エピソードの詳細は使用しない
- 「治療」「治癒」「医療」などの誤解を招く表現は使わない
- 過度なセクシャルアピールはせず、「癒し・安らぎ・信頼」を前面に出す
- 競合他店への批判・比較は行わない

## 出力例

```
【タイトル】
春の夕暮れに、ただひとりの時間を。

【本文】
今日の東京は、やわらかな春の空気に包まれていました。

施術が終わり、お客様をお見送りした後、しばらく窓の外を眺めていました。
夕陽が街をオレンジ色に染めていて、なんとも言えない静けさが漂っていた。

今日いらしてくださったお客様は、久しぶりの再会でした。
「仕事がずっと忙しくて、やっと来られた」と、おっしゃっていました。

特別なことは何もしていません。ただ、その方の今に寄り添うこと。
ゆっくりと呼吸が整っていくのを感じながら、時間が静かに流れていきました。

私がこの仕事を続けている理由のひとつは、こういう瞬間のためなのかもしれない。
日常の忙しさや疲れを、少しだけ手放せる場所でいたい。
そう、改めて思う春の夕暮れでした。

また次のひとときを、ご一緒できますように。

【投稿メモ】
- 最適な投稿タイミング：金曜夜 20時〜22時
- おすすめハッシュタグ：#女性用風俗 #リラクゼーション #癒し #東京
```
