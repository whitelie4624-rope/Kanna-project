# KANNA リピート促進メッセージSkill

## 概要
KANNAへのリピート来店を促すDM・フォローアップメッセージを生成するSkillです。感謝→次回提案→予約誘導の構成で200字以内のDM文を複数バリエーションで提案します。押しつけがましくならない自然な誘導文を心がけます。

## KANNAについて
- **サービス種別**：女性用風俗（女風）・リラクゼーション
- **ターゲット**：20〜40代女性
- **場所**：東京
- **トーン**：上品・温かみ・信頼感・非日常感
- **重視すること**：リピート率・顧客との信頼関係・ブランド一貫性

## 手順

### Step 1：状況の確認
以下の情報を確認する：
- 来店からの経過日数（3日後 / 1週間後 / 2週間後 / 1ヶ月後）
- 来店回数（初回後フォロー / 2回目以降のリピーター）
- 特別に触れたいこと（カルテのメモ・施術での気づきなど）
- キャンペーン・特典の有無

### Step 2：メッセージのタイプ選択
状況に応じて以下のタイプから最適なメッセージを設計する：
- **来店直後フォロー（3〜7日後）**：感謝＋体の状態確認
- **リピート誘導（2週間〜1ヶ月後）**：温かみのある再来店提案
- **特典・キャンペーン告知**：特典を絡めたリピート促進
- **季節・イベントメッセージ**：季節感を取り入れた自然な連絡

### Step 3：3段構成でメッセージ生成
1. **感謝**：来店への感謝（30〜50字）
2. **次回提案**：次回来店への自然な誘導（80〜100字）
3. **予約誘導**：行動を促す一言（20〜40字）

### Step 4：複数バリエーションの提案
同じテーマで3つの異なるバリエーションを提案する：
- バリエーション①：カジュアル・親しみやすい文体
- バリエーション②：上品・丁寧な文体
- バリエーション③：簡潔・スッキリした文体

## 出力フォーマット

```
## リピート促進DM（[来店経過日数 / タイプ]）

### バリエーション①（カジュアル）
[200字以内のメッセージ本文]
字数：○○字

### バリエーション②（上品・丁寧）
[200字以内のメッセージ本文]
字数：○○字

### バリエーション③（簡潔）
[200字以内のメッセージ本文]
字数：○○字

---
送信タイミング：[推奨の送信タイミング]
備考：[使用上のアドバイス]
```

## 注意事項
- 「また来てください」のような義務感・プレッシャーを与える表現は使わない
- 顧客のプライバシー（施術内容・相談内容）に触れる表現は避ける
- 「予約はこちら」のような直接的すぎる誘導は文脈に合わせて使う
- 送りすぎに注意：同じ顧客への連続DMは逆効果になりやすい
- 「女風」「性的サービス」を連想させる表現は使わない

## 出力例

```
## リピート促進DM（来店から2週間後 / リピート誘導）

### バリエーション①（カジュアル）
先日はご来店ありがとうございました。
お体の調子はいかがでしょうか？
あの日のゆったりした時間を、またご一緒できたら嬉しいです。
ご都合よければ、またいつでも声をかけてくださいね。
字数：88字

### バリエーション②（上品・丁寧）
先日は足をお運びいただきまして、誠にありがとうございました。
その後、お体の具合はいかがでしょうか。
またゆっくりとお時間をご一緒できる日を、楽しみにお待ちしております。
お気が向いた際は、どうぞお気軽にご連絡くださいませ。
字数：104字

### バリエーション③（簡潔）
先日のご来店、ありがとうございました。
またお体が疲れたときは、いつでも。
お待ちしています。
字数：44字

---
送信タイミング：来店から10〜14日後がベスト
備考：バリエーション②が最もKANNAのブランドイメージに合う。
      バリエーション③はリピーターへの軽い連絡に適している。
```
