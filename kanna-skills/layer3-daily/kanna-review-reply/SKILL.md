# KANNA 口コミ返信Skill

## 概要
KANNAへの口コミ・レビューに対して、KANNAのブランドトーンに合った返信文を生成するSkillです。ポジティブ・ネガティブ両方の口コミに対応し、感謝→共感→次回来店誘導の3段構成で150〜250字の返信文を生成します。

## KANNAについて
- **サービス種別**：女性用風俗（女風）・リラクゼーション
- **ターゲット**：20〜40代女性
- **場所**：東京
- **トーン**：上品・温かみ・信頼感・非日常感
- **重視すること**：リピート率・顧客との信頼関係・ブランド一貫性

## 手順

### Step 1：口コミの内容確認
ユーザーから口コミ文をコピーペーストしてもらう。
以下の情報も確認する：
- 口コミの種類（ポジティブ / ネガティブ / 中立）
- 掲載媒体（予約サイト / Google / SNS）
- 特に口コミで言及されているポイント

### Step 2：ポジティブ口コミへの返信
以下の構成で返信文を生成する：
1. **感謝**：口コミへの感謝を具体的に伝える
2. **共感・復唱**：口コミで良いと言ってくれたポイントを引用して共感する
3. **次回来店誘導**：次回の来店を自然に促すひと言

### Step 3：ネガティブ口コミへの返信
以下の構成で返信文を生成する：
1. **感謝**：貴重な意見をくださったことへの感謝
2. **謝罪・共感**：ご不満に対する誠実な謝罪と共感
3. **改善への意志**：どのように対応・改善するかを誠実に伝える
（次回来店誘導は文脈に応じて含める・含めない）

### Step 4：文体の調整
- 敬語を使いながらも、温かみのある親しみやすい文体
- 定型文・コピペ感を出さない
- KANNAらしい「癒し・信頼感」が伝わる言葉選び

## 出力フォーマット

**ポジティブ口コミへの返信：**
```
[感謝の言葉（30〜50字）]
[口コミポイントへの共感（60〜100字）]
[次回来店誘導（30〜50字）]
合計：150〜250字
```

**ネガティブ口コミへの返信：**
```
[感謝の言葉（30〜50字）]
[謝罪・共感（70〜120字）]
[改善の意志（30〜60字）]
合計：150〜250字
```

## 注意事項
- 口コミに記載されている施術の詳細な内容を返信文で繰り返さない
- 「女風」「性的サービス」を連想させる表現は使わない
- ネガティブ口コミに対して言い訳や反論はしない
- 顧客を特定できる情報（来店日・名前など）は返信文に含めない
- 過度に謙遜しすぎず、KANNAのブランド価値を守った文体にする

## 出力例

### ポジティブ口コミ例
**口コミ（入力）：**
「初めて利用しました。緊張していたのですが、話しかけてくださって
とてもリラックスできました。また来たいと思います！」

**返信文（出力）：**
はじめてのご来店、ありがとうございました。
緊張される中、足を運んでくださったこと、とても嬉しく思います。
少しでもほぐれていただけたなら、こちらこそ幸せです。
またお会いできる日を、楽しみにお待ちしております。
どうぞお気軽にいらしてくださいね。

ご来店いただきましてありがとうございます。

深くお詫び申し上げます。
今後は予約確認のご連絡をより丁寧に行ってまいります。
貴重なご意見をいただき、ありがとうございました。
またご縁がございましたら、ぜひお声がけください。
