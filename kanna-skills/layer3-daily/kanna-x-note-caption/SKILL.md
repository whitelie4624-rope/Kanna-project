# KANNA X・note投稿文Skill

## 概要
KANNAのX（旧Twitter）とnote向けの投稿文を生成するSkillです。X用（140字以内）またはnote用（冒頭400〜600字）の投稿文を生成します。Xはハッシュタグ2〜3個、noteはSEOを意識したタイトルと冒頭文を含みます。

## KANNAについて
- **サービス種別**：女性用風俗（女風）・リラクゼーション
- **ターゲット**：20〜40代女性
- **場所**：東京
- **SNS**：Instagram・X（旧Twitter）・note
- **トーン**：上品・温かみ・信頼感・非日常感
- **セラピスト**：男性セラピスト（オーナー兼任）

## 手順

### Step 1：媒体とテーマの確認
以下を確認する：
- 媒体（X / note / 両方）
- 投稿テーマ（日記 / 施術の話 / 考え方・コラム / お知らせ / 季節ネタ）
- 伝えたい内容のキーワード・メインメッセージ
- noteの場合：記事全体のテーマと想定読者

### Step 2：X用投稿文の生成
**特性：**
- 140字（全角）以内の短いテキスト
- ハッシュタグは2〜3個のみ（多すぎるとスパム感が出る）
- 最初の一文でリプライ・いいねを誘発する
- 「共感」「癒し」「気づき」系のコンテンツが拡散されやすい

**生成パターン：**
- パターンA：日記・つぶやき系（共感・温かみ）
- パターンB：お知らせ・告知系（簡潔・行動喚起）
- パターンC：考え方・コラム系（気づき・学び）

### Step 3：note用冒頭文の生成
**特性：**
- SEOを意識したタイトル（28〜32字、検索キーワードを含む）
- 冒頭400〜600字で読者を引き込む
- 「なぜこの記事を書いたか」「誰に読んでほしいか」を明示する
- noteは長文コンテンツなので、続きを読みたくなる冒頭文にする

**SEOキーワード例：**
- 「女性用風俗 東京」「リラクゼーション 一人で」「女性 マッサージ 信頼できる」
- 「疲れた 癒し 東京」「30代 自分へのご褒美」

### Step 4：バリエーション提案
X用：2〜3バリエーション
note用：タイトル3案 + 冒頭文1本

## 注意事項
- X（旧Twitter）では特にリアルタイム性・人間味が重要
- noteはSEO対策とブランディングの両立を意識する
- 両媒体とも性的表現・女風を直接的に表現する言葉は避ける
- noteのタイトルに「女性用風俗」を含む場合は、検索意図に合った内容にする
- Xのつぶやきは「セラピスト個人の人間味」が出るコンテンツが効果的
